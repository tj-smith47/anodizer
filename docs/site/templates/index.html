Release engineering for Rust, simplified.

The declarative release pipeline GoReleaser users wish existed for Rust.

Get Started GitHub 
$ cargo install anodize
$ anodize init          # generates .anodize.yaml from Cargo.toml
$ anodize release       # build → archive → checksum → release → publish

Full Pipeline

Build → Archive → Checksum → Changelog → Release → Publish → Announce. One command.

Cargo-Native

Workspace-aware, cross-compilation via cargo-zigbuild, cargo-binstall metadata generation.

Familiar Config

Same YAML structure GoReleaser users already know. Migrate in minutes.

Tera Templates

Conditionals, pipes, filters — a real template engine, not regex substitution.

Package Managers

Homebrew, Scoop, crates.io, nFPM, Docker — publish everywhere from one config.

Single Binary

cargo install anodize. No runtime dependencies. Fast startup.
